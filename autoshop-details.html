Project information

  • Category: Web development
  • Client: RT Auto Works.
  • Tools: HTML, CSS, JavaScript, PHP and MySQL.
  • Project URL: www.sharptechdev.com

Project description

Local mechanic/auto body shop needed assistance with information organization, prior to this form customer and vehicle information was manually stored and often written down on paper. Information would sometimes be outdated affecting the accuracy of parts delivery from outside vendors, along with being a security concern if left unsecured. This test from was created to display how information could be stored and retrieved on a database in order to be utilized when needed. In completing this project HTML, CSS, JavaScript was used on the frontend, PHP on the backend/server side and MySQL for the database.
