Project description

This welcome page was done as a demonstration for the courier based company Liberty Messenger Inc. The current website was outdated and the company was seeking to modernize the site to improve their service and attract new customers. Utilizing the organizations colour themes, a front facing demo page was created using HTML, CSS, JavaScript and Bootstrap.
